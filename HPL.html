3. HTML Lists

<!-- Unordered List -->
<ul>
    <li>Apple</li>
    <li>Banana</li>
    <li>Orange</li>
</ul>

<!-- Ordered List -->
<ol>
    <li>Wake Up</li>
    <li>Study</li>
    <li>Sleep</li>
</ol>

Preview

Unordered List

  • Apple
  • Banana
  • Orange

Ordered List

  1. Wake Up
  2. Study
  3. Sleep
Tag Definition
<ul> Creates an unordered (bulleted) list.
<ol> Creates an ordered (numbered) list.
<li> Represents an individual list item inside a <ul> or <ol>.
