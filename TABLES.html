HTML Tables Tutorial

Code Example

<table>

    <caption>Student Marks Report</caption>

    <thead>
        <tr>
            <th>ID</th>
            <th>Name</th>
            <th>Marks</th>
        </tr>
    </thead>

    <tbody>
        <tr>
            <td>101</td>
            <td>Alice</td>
            <td>95</td>
        </tr>

        <tr>
            <td>102</td>
            <td>Bob</td>
            <td>88</td>
        </tr>

        <tr>
            <td>103</td>
            <td>Charlie</td>
            <td>91</td>
        </tr>
    </tbody>

    <tfoot>
        <tr>
            <td colspan="2">Average Marks</td>
            <td>91.3</td>
        </tr>
    </tfoot>

</table>

Definitions

Tag / Attribute Description
<table> Creates a table.
<caption> Adds a title or heading for the table.
<thead> Groups the header rows of the table.
<tbody> Groups the main data rows.
<tfoot> Groups the footer or summary rows.
<tr> Defines a table row.
<th> Defines a header cell.
<td> Defines a standard data cell.
colspan Makes one cell span multiple columns.
rowspan Makes one cell span multiple rows.
