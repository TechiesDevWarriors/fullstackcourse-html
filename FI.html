Definitions

Element Purpose
<form> Creates a form for collecting user input.
<label> Displays a label for an input field.
<input type="text"> Accepts single-line text.
<input type="email"> Accepts email addresses.
<input type="password"> Hides entered characters.
<input type="number"> Accepts numeric input.
<input type="date"> Allows date selection.
<input type="radio"> Lets users select one option from a group.
<input type="checkbox"> Lets users select multiple options.
<select> Creates a drop-down list.
<option> Represents an item inside a drop-down list.
<textarea> Accepts multi-line text input.
<button> Creates clickable buttons such as Submit or Reset.
