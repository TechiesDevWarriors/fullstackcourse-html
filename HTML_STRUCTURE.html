📘 HTML Structure & Semantic HTML Guide

1. Basic HTML Structure (Code)

<!DOCTYPE html>
<html lang="en">

    <head>
        <meta charset="UTF-8">
        <meta name="viewport"
              content="width=device-width, initial-scale=1.0">
        <title>My Web Page</title>
    </head>

    <body>
        <h1>Welcome</h1>
        <p>Hello World</p>
    </body>

</html>

2. Visual Preview of Basic HTML Structure

<!DOCTYPE html>
<html>
<head>
  • <meta charset="UTF-8">
  • <meta name="viewport">
  • <title>My Web Page</title>
</head>
<body>
  • <h1>Welcome</h1>
  • <p>Hello World</p>
</body>
</html>

The browser reads the document from DOCTYPE → html → head → body. Only the content inside the <body> is displayed on the webpage.

Tag Definition
<!DOCTYPE html> Declares that the document uses HTML5.
<html> The root element that wraps the entire HTML document.
<head> Contains metadata, styles, scripts, and the page title.
<meta charset="UTF-8"> Specifies UTF-8 character encoding.
<meta viewport> Makes the page responsive on mobile devices.
<title> Sets the text displayed on the browser tab.
<body> Contains everything visible to the user.

3. Semantic HTML Structure (Code)

<header>
    <h1>Website</h1>
    <nav>Navigation</nav>
</header>

<main>

    <section>
        General Content
    </section>

    <article>
        Blog Post
    </article>

    <aside>
        Sidebar
    </aside>

</main>

<footer>
    Copyright
</footer>

4. Visual Preview of Semantic HTML

<header>
<nav> Navigation Menu </nav>
<main>
<section>

General page content.

</section>
<article>

Independent article or blog post.

</article>
<aside>

Sidebar or related links.

</aside>
</main>
<footer> © 2026 My Website </footer>
Semantic Tag Definition
<header> Represents introductory content like logos, titles, or navigation.
<nav> Contains navigation links for the website.
<main> Represents the primary content of the page.
<section> Groups related content into a logical section.
<article> Represents self-contained content such as a blog post or news article.
<aside> Contains supplementary information like sidebars or related links.
<footer> Represents the footer of a page or section.

5. Basic HTML vs Semantic HTML

Basic HTML Semantic HTML
Uses generic tags such as <div>. Uses meaningful tags such as <header>, <main>, and <footer>.
Mainly defines structure. Defines both structure and meaning.
Less descriptive for accessibility and search engines. Improves accessibility and SEO.
